The 4-Layer Clean System (STRICT DIRECTION)Dependency direction is strictly: Routes → Services → Repositories → DB.Layer 1: Routes (app/api/): HTTP endpoints ONLY. Zero business logic. Orchestration of multiple repositories.Layer 3: Repositories (app/repositories/): The ONLY place for prisma.* queries. CRUD only.Layer 4: DB (Supabase/Postgres): Managed via Prisma.3.
