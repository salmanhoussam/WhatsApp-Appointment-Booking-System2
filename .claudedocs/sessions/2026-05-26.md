## 🔧 Technical Decisions Made

- **Zero-rerender bridge:** `ScrollProgressContext` يمرر `MotionValue` (لا قيمة) إلى R3F Canvas. `useFrame(() => scroll.get())` يقرأ بدون subscription → صفر re-renders على الـ scroll
- **Shader approach:** `shaderMaterial` من drei + `extend({ SlotMat })` → JSX `<slotMat />` مع uniform `uOpacity` مباشرة على instance. Vignette: `smoothstep(0.26, 0.50, length(d * vec2(1.0, 1.35)))`
- **ai-agent-canvas pattern:** SLOTS هو JSON config خارج الـ component — agent أو admin يعدّل JSON والـ Canvas يرسم تلقائياً
- **Position calculation:** Camera z=5, FOV=55 → viewport half-width ≈ 4.33 units. Slots عند x=3.8+ تخرج من الشاشة → صُحّحت إلى x=1.9–2.5
- **CSS olive div removed:** نظيفة الآن، الـ R3F يتولى الـ olive بالكامل
