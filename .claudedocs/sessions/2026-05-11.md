### 3 أنماط الـ Hero (`page_type`)

| # | الاسم | `page_type` | الوصف |
|---|---|---|---|
| T1 | **Minimal** | `normal` | الحالي — dark glass، color orb، centered title، CTA button |
| T2 | **Showcase** | `showcase` | Split-screen — نص يسار، صورة/animated rings يمين |
| T3 | **Landing** | `landing` | جديد — full-screen gradient، headline كبير، floating particles CSS، horizontal catalog strip |

**التغيير في ConfigurableHero.jsx:**
```jsx
// أضف LandingHero component جديد
function LandingHero({ config }) { ... }

export default function ConfigurableHero({ config }) {
  if (config?.page_type === 'showcase') return <HeroErrorBoundary><ShowcaseHero /></HeroErrorBoundary>
  if (config?.page_type === 'landing')  return <HeroErrorBoundary><LandingHero  /></HeroErrorBoundary>
  return <SimpleHero config={config} />
}
```

**LandingHero spec:**
- Background: `radial-gradient` متحرك من اللون الرئيسي (نقطتان) + noise texture CSS
- Headline: `clamp(48px, 8vw, 96px)`, fontWeight 900, letterSpacing -0.06em, RTL
- Sub-headline: صغير، opacity 0.5
- CTA: pill button مع glow shadow كبير
- Bottom: catalog strip أفقي يظهر تدريجياً (fade in من الأسفل)
- Animation: spring stiffness 60 damping 20
