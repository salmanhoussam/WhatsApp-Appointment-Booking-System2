### 🟡 Task 2 — إصلاح Race Condition في `service_service.py` + `price_service.py`

**المشكلة:** `update` و `delete` يتحققان من `clientId` أولاً عبر `get_*()` ثم يُنفّذان العملية بدون `clientId` في الـ `where`:

```python
# الحالي ❌ — إذا حُذف الـ record بين السطرين → Prisma error بدل 404 نظيف
existing = await get_service(db, service_id, client_id)  # يتحقق هنا
if not existing: return None
await db.service.update(where={"id": service_id}, data=update_data)  # بدون clientId
```

**الإصلاح في `app/services/service_service.py`:**

```python
# update_service — أضف clientId للـ where
return await db.service.update(
    where={"id": service_id, "clientId": client_id},  # ← أضف هذا
    data=update_data
)

# delete_service — أضف clientId للـ where
await db.service.delete(
    where={"id": service_id, "clientId": client_id}   # ← أضف هذا
)
```

**الإصلاح في `app/services/price_service.py`:**

```python
# update_price
return await db.price.update(
    where={"id": price_id, "clientId": client_id},    # ← أضف هذا
    data=update_data
)

# delete_price
await db.price.delete(
    where={"id": price_id, "clientId": client_id}     # ← أضف هذا
)
```

> **ملاحظة Prisma:** تأكد أن `clientId` مش جزء من `@@id` compound — إذا كان كذلك استخدم أسلوب آخر.

#### 7.3 — Frontend: `useImageUpload.js` (Phase 52.5)

**الملف الجديد:** `frontend/src/hooks/useImageUpload.js`

```js
// Hook يُستخدم في CatalogTab + SettingsTab
// يرفع صورة → يرجع URL + imageId

export function useImageUpload() {
  const [uploading, setUploading] = useState(false)
  const [error, setError]         = useState(null)

  const upload = async (file, context, contextIds = {}) => {
    setUploading(true)
    const form = new FormData()
    form.append('file', file)
    form.append('context', context)
    Object.entries(contextIds).forEach(([k, v]) => form.append(k, v))

    try {
      const res = await adminApi.post('/upload', form, {
        headers: { 'Content-Type': 'multipart/form-data' }
      })
      return res.data  // { url, image_id }
    } catch (e) {
      setError(e.message)
      return null
    } finally {
      setUploading(false)
    }
  }

  return { upload, uploading, error }
}
```
