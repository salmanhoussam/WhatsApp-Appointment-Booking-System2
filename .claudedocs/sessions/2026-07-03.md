## 🔧 Technical Decisions Made

- **`python -m prisma generate` لا `npx prisma generate`**: Node CLI version يختلف عن Python client — Python CLI يطابق تلقائياً
- **ASCII فقط في schema comments**: Windows cp1252 لا يدعم box-drawing chars (═ U+2550) — يسبب surrogate error في Python generator
- **Migration عبر Prisma لا REST API**: new project's PostgREST يرجع 403 "permission denied for schema public" — Prisma (DIRECT_URL) شغّال بالكامل
- **Old Supabase via REST API لا direct DB**: `db.gdzthjcvzvhfpsvoxhbm.supabase.co:5432` لا يحلّ DNS من هذا الجهاز؛ HTTPS يشتغل عادي
- **Images: re-upload إلى new bucket**: نقلنا كل الصور من `menu/` bucket القديم لـ `properties/{slug}/catalog/{cat_id}/{item_id}/main.ext` في الجديد؛ external URLs (Unsplash) تركناها كما هي
