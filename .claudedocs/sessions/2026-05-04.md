#### App.jsx — التغييرات (52.6)

```jsx
// أُضيف:
const GenericAdminDashboard = lazy(() => import('./pages/generic-admin/GenericAdminDashboard'));
const TenantRegisterPage    = lazy(() => import('./pages/auth/TenantRegisterPage'));

// /:slug/dashboard — كان SmarAdminDashboard لكل التيننتس:
<Route path="/:slug/dashboard/*" element={
  <ProtectedRoute>
    <Suspense fallback={null}><GenericAdminDashboard /></Suspense>
  </ProtectedRoute>
} />

// /register — أصبح يكشف ?template= param:
<Route path="/register" element={
  new URLSearchParams(window.location.search).has('template')
    ? <Suspense fallback={null}><TenantRegisterPage /></Suspense>
    : IS_AUTH_SUBDOMAIN
      ? <Suspense fallback={null}><SSOLoginPage /></Suspense>
      : <Navigate to="/" replace />
} />
```

**القرار:** `?template=` يُفحص في route element (مش module scope) — يتقيّم عند كل navigation، مش مرة وحدة.

#### TenantRegisterPage.jsx (52.1)

**Flow التسجيل — 5 خطوات:**

```
1. POST /api/v1/auth/register
   → { token (CLIENT JWT), slug, client_id }

2. POST /api/v1/auth/users/login
   → { token (USER JWT) } → يُخزَّن في localStorage('admin_access_token')

3. PATCH /api/v1/admin/settings
   → { name_ar, primary_color, template_key, page_type }

4. POST /api/v1/admin/catalog/seed-from-template   (إذا template.seedCategories.length > 0)
   → يزرع التصنيفات

5. navigate(`/${form.slug}/dashboard?welcome=1`, { replace: true })
```

**تفاصيل:**
- يقرأ URL params: `?template=`, `?color=`, `?slug=`
- Auto-slugify: `business_name` → `slug` (lowercase + hyphens)
- `authApi` = `axios.create({ baseURL: API_BASE })` — منفصل عن `adminApi` (لـ /auth/*)
- Layout: عمودين — يسار form، يمين preview القالب + swatch + seed categories chips
- Error handling: إذا Step 3 أو 4 فشلت → يكمل للـ dashboard (التسجيل نجح)
